## Key Design Decisions & Lessons Learned

### What Works (Keep These)
- Neon dark theme (--neon-green: #39ff14, --neon-blue: #00f0ff, --dark-bg: #0a0a1a)
- Orbitron font for headers, Rajdhani for body, Space Mono for data
- Single-file HTML approach (zero dependencies, instant load)
- Cursor-following particle effects (subtle, max 15-20 particles)
- Educational facts shown during gameplay, not just at end
- Real EIE data (cal/$, protein/$, time savings)

### What Doesn't Work (Avoid)
- @import url() in CSS — ALWAYS use <link> tags instead. @import after :root breaks fonts.
- Memory card matching games — users find them boring
- Plain snake without unique mechanics — add growth zones, bosses, combos
- Small text on game elements — minimum 12px for any readable text
- Assuming emoji will render on all systems — use safe/common emoji only
- Full file rewrites in sub-agents — SURGICAL EDITS with edit tool for files >30KB

### Critical Bugs Found & Fixed
1. optimizer-quest.html: Malformed unicode `\u{1FA D5}` (space in hex) broke entire JS
2. optimizer-quest.html: @import after :root in CSS broke font loading — converted to <link>
3. budget-blaster.html: @import outside <style> tag rendered as visible white text
4. budget-blaster.html: Mouse attraction applied when idle, causing constant upward drift
5. nutrient-tetris.html: "DlvyFee" with \u{1F6D1} emoji rendering as "DOTS" on some systems

### User Feedback Themes
- EDUCATION is critical — every game must teach real EIE data
- Fun > Education — but education should be woven in, not bolted on
- Progression matters — games need to get harder and reward continued play
- Text must be readable — especially on mobile
- Show correct answers — when quizzes/challenges are wrong, ALWAYS show why
- Incentivize exploration — Efficiency Empire's "Collection Bonus" forces buying all upgrades
- Visual feedback — particles, screen shake, combos, floating text all make games feel alive

### Kitchen Alchemist — Science Audit Results
VERIFIED CORRECT:
- MSG is safe (FDA GRAS classification)
- Salt early for deeper seasoning (osmosis)
- Acid brightens at end of cooking (preserves volatiles)
- Fat carries flavor (fat-soluble compounds)
- High heat → Maillard reaction (280-330°F)

FIXED (was wrong):
- "Potatoes absorb salt" — MYTH (they absorb liquid, not selectively salt)
- Cold lemonade claim about ethanol vapor — lemonade has no ethanol; fixed to taste perception
- "MSG enhances flavor 10x" — removed unsubstantiated multiplier
- "Salt needed for blood pressure regulation" — backwards; salt suppresses bitterness
- "Citrus = Vitamin C your body craves" — actual mechanism is citric acid contrast
